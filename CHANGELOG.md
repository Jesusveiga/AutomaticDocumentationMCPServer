## [1.0.0] - 2026-05-13

### Added
- ✨ Initial release of Power BI Documentation Generator
- 📋 Automatic metadata extraction from `.pbip` and `.pbism` projects
- 🎨 Professional technical design document generation in Markdown format
- 📄 Markdown to DOCX to PDF conversion pipeline with corporate styling
- 🎯 Batch processing support for multiple models
- 📊 Complete measure catalog generation with DAX code
- 🏗️ Star schema architecture documentation
- 🔗 Relationship mapping with business logic
- 🔐 Security & governance documentation
- 💼 Professional Word template with corporate styling
- 🎓 Comprehensive documentation and setup guides
- 🤝 Contributing guidelines

### Features
- [x] Extract metadata from Power BI projects
- [x] Generate professional technical documentation
- [x] Apply corporate styling and formatting
- [x] Export to DOCX with table styling
- [x] Convert DOCX to PDF
- [x] Support for batch file processing
- [x] Automated measure documentation
- [x] Relationship documentation with cardinality

### Documentation
